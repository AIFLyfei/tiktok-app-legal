AI Fly Social Publisher

AI Fly Social Publisher is a social publishing service for creating, scheduling, and distributing short-form content across supported platforms. This is the public product homepage — no login is required to view this page.

Service status: Active development and compliance review.

Contact: aiflyfei@gmail.com

What this product does

1) Connect

Users connect their TikTok account via TikTok's official OAuth authorization flow. No credentials are collected directly.

2) Create & Schedule

Users upload media, add captions, configure metadata, and schedule a publish time.

3) Publish

The service submits content to TikTok on behalf of the authorized user using the Content Posting API.

4) Track

Basic publish status and diagnostics are stored to help users verify delivery and troubleshoot failures.

TikTok API Products & Scopes Used

The following scopes are requested from users. Only the scopes listed below are used; no additional scopes are requested.

Scope / Permission Product How it is used
user.info.basic Login Kit Identify the authenticated TikTok user (display name, avatar) to associate their account in the app.
video.publish Content Posting API Upload and publish videos to the user's TikTok account as authorized by the user.
video.upload Content Posting API Upload video files to TikTok's servers as part of the two-step publish flow.

If any scope above is not yet fully implemented, it will be removed from the app configuration before the next resubmission.

Sandbox / Test Account

All integration testing is performed in the TikTok sandbox environment. The app uses the TikTok sandbox API endpoints during development and review.

To test the integration:

  1. Log in to the TikTok Sandbox using the sandbox test account credentials provided via the TikTok developer portal's sandbox user panel.
  2. Authorize the app through the standard TikTok OAuth flow.
  3. Use the in-app upload interface to select a video file, add a caption, and submit for publishing.
  4. Verify the video appears in the sandbox account's post list via the TikTok developer portal.

If a dedicated sandbox test account is required, please contact aiflyfei@gmail.com and one will be provided promptly.
